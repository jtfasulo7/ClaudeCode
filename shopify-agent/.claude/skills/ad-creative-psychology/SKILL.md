# Ad Creative Psychology & Sales Engine

## CRITICAL: Agent Execution Order

Every time you are asked to create ad creatives for a product, you MUST follow this
exact sequence. Do NOT skip steps. Do NOT jump to creative generation.

```
PHASE 1 → COMPETITIVE INTELLIGENCE (Research what's already winning)
PHASE 2 → CREATIVE AUTOPSY (Dissect WHY winning ads work)
PHASE 3 → PSYCHOLOGY-DRIVEN ANGLE SELECTION (Choose your attack vector)
PHASE 4 → CREATIVE BRIEF & PROMPT GENERATION (Build the brief, write the prompts)
```

## PHASE 1: COMPETITIVE INTELLIGENCE

### Step 1A: Find Every Competitor Ad for This Exact Product

Use the `pipeboard-meta-ads` MCP to search the Facebook Ad Library. Also search the
TikTok Creative Center (ads.tiktok.com/business/creativecenter) manually.

**Search strategy (execute ALL of these):**
1. Search the exact product name
2. Search the product category (e.g., "posture corrector", "red light therapy")
3. Search 3-5 competitor brand names selling the same or similar products
4. Search benefit keywords (e.g., "back pain relief", "better sleep")
5. Search the product on CJ Dropshipping and find other stores selling it

**For each ad found, log:**
- Advertiser name
- Ad format (image / video / carousel / collection)
- How long it has been running (CRITICAL — longer = more profitable)
- Number of ad variations (more variations = active testing = serious spender)
- Platform (Facebook Feed, Instagram Reels, Stories, etc.)
- Primary text / headline / CTA
- Visual style (UGC, studio, lifestyle, product-on-white, demo, etc.)

### Step 1B: Identify the Winners

**Proxy signals for high-performing ads (since Ad Library doesn't show metrics):**

| Signal | What It Means |
|--------|--------------|
| Running 30+ days | Almost certainly profitable — advertisers kill losers fast |
| Running 60+ days | Proven winner, likely scaled to high spend |
| 5+ active variations | Brand is actively split-testing — serious media buyer |
| Multiple platforms simultaneously | Scaled across placements = strong ROAS |
| Ad has been refreshed/iterated | Creative team found a winner and is extending its life |
| Similar angle used by 3+ competitors | Market-validated angle — the audience responds to this |

**Rank the top 5-10 winning ads** by run duration × variation count.

### Step 1C: Determine Image vs Video Dominance

Count the format distribution across ALL winning ads (30+ day runners):

- If >65% are video → This product sells through motion/demo/story. Plan VIDEO.
- If >65% are static image → This product sells on visual impact alone. Plan IMAGE.
- If mixed → Plan BOTH but lead with the format used by the #1 longest-running ad.

**Health & wellness defaults (from industry data):**
- Supplements: UGC video dominates (32.6% UGC, then studio product shots)
- Fitness equipment: Demo video + before/after images
- Skincare/beauty: Creator-led video with ingredient callouts
- Sleep/recovery: Lifestyle imagery + soft cinematic video
- Pain relief devices: Problem→solution demo video almost always wins

## PHASE 2: CREATIVE AUTOPSY — Dissecting Why Winners Work

For each of the top 5 winning ads, perform this full autopsy. If it is a VIDEO,
you MUST watch it fully before analyzing. Do not skim.

### 2A: The Hook Audit (First 3 Seconds / First Glance)

**For video ads, identify which hook archetype is used:**

| Hook Type | Example | Why It Works |
|-----------|---------|-------------|
| Pain-point callout | "Still waking up with back pain?" | Activates problem-awareness, triggers amygdala |
| Bold/contrarian claim | "Doctors have been wrong about posture" | Curiosity gap, pattern interrupt |
| Transformation reveal | Shows result first, then rewinds | Dopamine anticipation, "how did they do that?" |
| Social proof lead | "2 million sold" or UGC unboxing | Consensus heuristic (Cialdini), reduces skepticism |
| Reverse psychology | "Don't buy this unless..." | Reactance theory, forbidden fruit effect |
| Question hook | "Why do Japanese people live longer?" | Opens a cognitive loop the brain needs to close |
| Demonstration | Product in action from frame 1 | Immediate relevance filter — shows exactly what it does |
| Shock/pattern interrupt | Unexpected visual or sound | Orienting response — involuntary attention capture |

**For image ads, identify the visual hook:**
- Contrast (before/after split)
- Human face with emotional expression (mirror neurons)
- Product hero shot with bold overlay text
- Lifestyle aspiration (the person you want to become)
- Infographic/educational (ingredient callouts, "how it works")
- Social proof composite (reviews/stars/numbers overlaid)

**Log the exact hook used and WHY it would stop a scroll.**

### 2B: The Emotional Architecture

Map the ad to its PRIMARY emotional driver. Every successful ad triggers one
dominant emotion. Identify it:

| Emotion | How It Manifests in Ads | Neurological Mechanism |
|---------|------------------------|----------------------|
| **Fear of missing out (FOMO)** | Scarcity, countdown, "selling fast" | Loss aversion — losing hurts 2x more than gaining feels good |
| **Aspiration/desire** | Lifestyle imagery, "imagine yourself..." | Dopamine anticipation pathway, future-self projection |
| **Pain/frustration relief** | "Tired of...?", problem agitation | Amygdala activation → urgency to resolve threat |
| **Social belonging** | "Join 100k+ customers", tribe language | Unity principle (Cialdini), in-group bias |
| **Curiosity** | Information gap, "the secret to...", "doctors don't want you to know" | Cognitive itch — the brain MUST close open loops |
| **Vanity/self-image** | Transformation, attractiveness, status | Medial prefrontal cortex — self-relevant processing |
| **Trust/safety** | Doctor endorsements, certifications, ingredients | Authority principle (Cialdini), risk reduction |
| **Instant gratification** | "Feel the difference in 7 days" | Reward system — dopamine release from anticipated pleasure |

### 2C: The Persuasion Stack

Identify which of Cialdini's 7 Principles are deployed in each ad.
Most winning ads stack 3-4 principles simultaneously:

1. **Reciprocity** — Free guide, free shipping, bonus item, valuable content
2. **Commitment/Consistency** — Quiz, "are you the type who...", small yes → big yes
3. **Social Proof** — Reviews, testimonials, user count, UGC, star ratings
4. **Authority** — Doctor/expert endorsement, clinical studies, certifications
5. **Liking** — Attractive/relatable creator, humor, shared identity
6. **Scarcity** — Limited stock, limited time, exclusive deal, seasonal
7. **Unity** — "We" language, tribe identity, shared values ("fellow health nuts")

**Log the specific persuasion stack for each winning ad.**

### 2D: The Conversion Architecture

Map the full structure of the ad:

**Video structure (most common winning patterns):**
```
HOOK (0-3s) → Problem agitation (3-7s) → Product intro as solution (7-12s) →
Demo/proof (12-20s) → Social proof reinforcement (20-25s) → Offer + CTA (25-30s)
```

OR for UGC-style:
```
HOOK (0-3s) → Personal story/problem (3-10s) → Discovery of product (10-15s) →
Usage demo (15-22s) → Results/transformation (22-27s) → Recommendation + CTA (27-30s)
```

**Image structure:**
```
Visual hook (hero image) + Headline (primary benefit or pain point) +
Body (2-3 supporting benefits or social proof) + CTA button
```

### 2E: Platform-Specific Patterns

**Meta (Facebook/Instagram) patterns that win:**
- 4:5 vertical video with bold black-on-white captions
- Product PNG overlays mid-video to anchor brand identity
- "Drama set-up" scripts: relatable problem → product resolution
- Interview/testimonial style — mimics organic social proof
- Magnifying glass animations on ingredients/features (wellness)
- Masking effects — obscure then reveal product feature
- Notification bar / simulated search bar graphics (native feel)

**TikTok patterns that win:**
- Lo-fi, phone-shot UGC that blends into FYP (NOT polished studio)
- Sound-on design: voiceover, trending audio, or ASMR
- Creator talking directly to camera (builds parasocial trust)
- #TikTokMadeMeBuyIt framing
- Reverse psychology hooks ("Don't buy this...")
- Problem → satisfying visual resolution (ASMR-adjacent)
- Face-to-camera with text overlay hooks
- Unboxing / first impressions format

## PHASE 3: PSYCHOLOGY-DRIVEN ANGLE SELECTION

### 3A: Choose the Winning Angle

Based on your Phase 2 autopsy, select ONE primary angle from this menu.
The angle you choose dictates everything — visuals, copy, emotion, structure.

**THE ANGLE MENU:**

| # | Angle | Best When | Emotion Lever | Example |
|---|-------|-----------|--------------|---------|
| 1 | **Problem-Agitate-Solve (PAS)** | Product fixes a painful problem | Fear → Relief | "Back pain ruining your sleep? This device fixed mine in 3 days" |
| 2 | **Before/After Transformation** | Visual results are dramatic | Aspiration + Proof | Split-screen or timeline transformation |
| 3 | **Us vs Them (Comparison)** | Product has clear advantages | Superiority + Logic | "Why I switched from [competitor] to this" |
| 4 | **Social Proof Avalanche** | Product has lots of reviews/sales | Trust + FOMO | Montage of reviews, unboxings, user clips |
| 5 | **Authority/Expert Endorsement** | Health/science claims matter | Trust + Authority | "As a physical therapist, I recommend..." |
| 6 | **Curiosity/Secret** | Product has a unique mechanism | Curiosity | "The Japanese secret to..." or "Why doctors are recommending..." |
| 7 | **Lifestyle Aspiration** | Product is an identity marker | Desire + Belonging | Show the elevated daily routine the product enables |
| 8 | **UGC Authentic Discovery** | Product is impulse-buy friendly | Trust + Relatability | "I saw this on TikTok and had to try it..." |
| 9 | **Scarcity/Urgency** | Running promotions or launches | FOMO + Loss aversion | "Only 200 left at this price" |
| 10 | **Educational/How-It-Works** | Product mechanism is its selling point | Curiosity + Authority | Animated explainer of how the technology works |
| 11 | **Emotional Storytelling** | Product has deep personal impact | Empathy + Connection | Customer tells their story of struggle → discovery → change |
| 12 | **Contrarian/Myth-Busting** | Audience has misconceptions | Curiosity + Authority | "Everything you've been told about [X] is wrong" |

### 3B: Build the Persuasion Stack for YOUR Ad

Choose exactly:
- 1 PRIMARY emotion (from the emotion table in 2B)
- 1 PRIMARY angle (from the angle menu above)
- 3-4 Cialdini principles to layer in
- 1 hook archetype (from the hook table in 2A)

**Example stack for a red light therapy panel:**
```
Primary emotion: Pain/frustration relief
Angle: Problem-Agitate-Solve with Authority
Cialdini stack: Authority (clinical studies) + Social Proof (reviews) + Scarcity (sale)
Hook: "Why are thousands of people replacing their pain meds with this light?"
```

### 3C: Validate Against the 5-Point Conversion Checklist

Before finalizing, confirm your angle hits ALL five:

- [ ] **STOP** — Will this stop the scroll in under 1.5 seconds?
- [ ] **FEEL** — Does this trigger a genuine emotional response?
- [ ] **BELIEVE** — Is there enough proof/authority to overcome skepticism?
- [ ] **WANT** — Does the viewer picture themselves using/benefiting from this?
- [ ] **ACT** — Is the next step (click/buy) frictionless and urgent?

If any box is unchecked, strengthen that element before proceeding.

### 4B: Nano Banana 2 Prompt Engineering (for images)

When generating the hero image with Nano Banana 2, structure the prompt to hit
the psychological levers identified above:

**Prompt formula:**
```
[Visual style], [subject + action], [product placement], [emotional atmosphere],
[lighting], [color palette], [composition], [platform-optimized aspect ratio]
```

**Example:**
```
Lifestyle photography, a woman in her 30s holding a red light therapy panel
against her lower back while sitting on a modern couch in a bright living room,
warm golden hour lighting, soft earth-tone palette, shallow depth of field with
product in sharp focus, relaxed and relieved expression, 4:5 vertical composition
for Instagram Feed
```

**Key rules for psychology-driven image generation:**
- ALWAYS include a human element (faces drive 38% more engagement)
- ALWAYS show the product IN USE, not just sitting there
- Match the EMOTIONAL STATE of the model to the desired viewer emotion
- Use warm lighting for trust/comfort, cool lighting for clinical/authority
- Leave space for text overlay hooks (top 20% of frame)
- Product should be clearly visible but not dominate > 40% of frame (feels like an ad)
- For before/after: left = problem (cooler tones, tension), right = solution (warm, relief)

### 4C: Kling Video Prompt Engineering (for video ads)

When generating the video with Kling 3.0, structure each shot to follow the
psychological architecture from Phase 3:

**Shot-by-shot template:**
```
SHOT 1 (Hook — 0-3s): [Dramatic visual that creates curiosity or pattern interrupt]
SHOT 2 (Problem — 3-7s): [Visual representation of the pain point]
SHOT 3 (Product intro — 7-12s): [Product reveal with key benefit text overlay]
SHOT 4 (Demo/proof — 12-20s): [Product in use, showing mechanism or result]
SHOT 5 (Social proof — 20-25s): [Review overlay, user count, or testimonial clip]
SHOT 6 (CTA — 25-30s): [Product beauty shot with offer and clear CTA]
```

**Kling-specific rules:**
- Use 9:16 for TikTok/Reels, 4:5 for Feed
- Generate 3-5 camera move variations (zoom, pan, static, dolly)
- First frame MUST be scroll-stopping (Kling generates from the first frame)
- Keep AI motion subtle and natural — uncanny motion kills trust
- For UGC-style: prompt for phone-camera perspective, slight shake, natural light
- For studio: prompt for smooth dolly, product-on-pedestal, clean background

### 4D: Ad Copy Psychology Frameworks

Write all ad copy using one of these proven frameworks:

**PAS (Problem-Agitate-Solve):**
```
[Problem] Tired of waking up with a stiff neck every morning?
[Agitate] You've tried new pillows, stretches, even chiropractor visits — nothing sticks.
[Solve] This cervical traction device realigns your spine in 10 minutes. 94% of users
report pain relief within the first week. Try it risk-free →
```

**AIDA (Attention-Interest-Desire-Action):**
```
[Attention] 2.3 million Americans discovered this back pain secret last year.
[Interest] It's a clinical-grade red light therapy panel you can use at home.
[Desire] Imagine waking up pain-free, without pills or doctor visits.
[Action] Get 40% off today only → Shop Now
```

**BAB (Before-After-Bridge):**
```
[Before] Spending $200/month on massage and still hurting.
[After] Pain-free mornings, $0/month in ongoing costs.
[Bridge] This FDA-cleared device changed everything. See why 50,000+ people switched →
```

**HOOK COPY FORMULAS (for primary text opening line):**
- "I was skeptical until [specific result happened]"
- "My [doctor/therapist/trainer] told me about this and I can't believe I waited"
- "Stop doing [common wrong approach]. Here's what actually works:"
- "[Number] people bought this last month. Here's why:"
- "This is the [product] that [specific community] won't stop talking about"

## PHASE 6: PRODUCT LISTING IMAGE GENERATION (Nano Banana 2)

When generating product listing images for Shopify, follow these rules:

### Image Set Requirements (5 per product)
1. **Front shot** — Product centered on dark premium surface, LEDs glowing, studio lighting
2. **Angle shot** — 3/4 view showing depth, straps, ports, construction details
3. **Detail/close-up** — Macro of LED array, material texture, key feature
4. **Model shot** — Woman 28-35 wearing the mask, spa/lifestyle setting, realistic skin
5. **Infographic** — Product on dark navy background with clean white callout labels pointing to features. NO morphed text — use simple short labels only.

### Critical Accuracy Rules
- Only generate images of the REAL product being sold. Reference the original supplier listing images.
- Entry/Mid tier masks: WHITE translucent rigid plastic, wired with controller, elastic strap
- Premium tier masks: BLACK flexible silicone, wireless rechargeable, no wires
- Never add features that don't exist on the real product
- Never change the product color, material, or shape
- The model shot must show the CORRECT mask (white or black depending on tier)
- Infographic labels must be factually accurate to the real product specs

### Nano Banana 2 Prompt Formula for Product Images
```
[Shot type] of [EXACT product description matching supplier listing],
[surface/environment], [lighting], [composition], [what LEDs are doing],
[any accessories visible: controller, wire, strap], [aspect ratio],
photorealistic product photography, no text unless infographic
```

### Rate Limiting
- 10 second delay between each Gemini generation
- Retry up to 3x on 429 errors with exponential backoff (20s, 40s, 60s)
- If model gemini-3.1-flash-image-preview is rate limited, fall back to gemini-2.5-flash-image
